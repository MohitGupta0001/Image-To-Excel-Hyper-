# 📊 Pytorch Image-to-Excel Pro

A premium AI-powered tool for extracting structured tables from images directly into Excel spreadsheets.

## 🚀 Powered by
- **PyTorch**: Deep learning backend.
- **Microsof Table Transformer (TATR)**: State-of-the-art table structure recognition.
- **Tesseract OCR**: High-fidelity cell content extraction.
- **Streamlit**: Modern, elegant user interface.

## 🛠️ Setup Instructions

1. **Install Tesseract OCR**:
   - Download and install from [Tesseract OCR Windows](https://github.com/UB-Mannheim/tesseract/wiki).
   - Ensure it's installed at `C:\Program Files\Tesseract-OCR\tesseract.exe`.

2. **Run the Application**:
   - Open PowerShell in this directory.
   - Run: `.\run.ps1`

## 📂 Features
- **Smart Detection**: Automatically finds tables within full-page document scans.
- **Precision Structure Recognition**: Identifies rows and columns with pixel-perfect accuracy.
- **Multi-Format Support**: Works with PNG, JPG, JPEG, and TIFF.
- **Visual Feedback**: See exactly what the AI 'sees' with detection overlays.
- **One-Click Export**: Instant Excel generation and download.
